GitHub Pages Deployment

This project demonstrates how to automatically deploy a static website to GitHub Pages using GitHub Actions.
The deployment happens only when changes are made to the index.html file.

Create a repository for the project.

Add a simple index.html file with a basic message.

Add a README.md explaining the project.

Create a workflow file inside .github/workflows to handle the deployment.

The workflow should:

Run on every push to the main branch.

Trigger only when the index.html file is updated.

Deploy the website to GitHub Pages automatically.
